DEKALB GARDENING LEGAL BENCHMARK
24 TEST PROMPTS AND ANSWER KEY
Companion file for the 10-document benchmark set

====================================================================
1. PURPOSE
====================================================================

This file provides a runnable 24-case benchmark for the DeKalb gardening
legal-style document set. It is designed for bounded efficacy testing of
a local legal document assistant. The benchmark focuses on:
- relevant passage retrieval
- grounded question answering
- targeted summarization / extraction

This benchmark is not intended to measure:
- open-ended legal reasoning
- legal advice quality
- research-grade model comparison
- comparison against cloud AI systems

====================================================================
2. SCORING FORMAT
====================================================================

Scoring labels:
- Pass
- Partial
- Fail

General guidance:
- Retrieval: Pass if the correct or clearly relevant section appears in
  the top returned results.
- Grounded QA: Pass if the answer is materially correct and supported by
  the source text.
- Summarization / Extraction: Pass if the response is useful, grounded,
  and includes the key facts without material fabrication.
